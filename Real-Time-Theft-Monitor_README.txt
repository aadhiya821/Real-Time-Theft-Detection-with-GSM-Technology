# 🚨 Real-Time Theft Monitoring System using GSM Module

## 📌 Project Overview
The Real-Time Theft Monitoring System is an embedded security solution designed to detect unauthorized access and instantly notify the owner through SMS using a GSM module. The system uses sensors connected to a microcontroller to monitor intrusion events, making it suitable for homes, offices, and small businesses.

---

## 🎯 Objectives
- Detect theft or unauthorized access in real time.
- Send instant SMS alerts to the registered mobile number.
- Provide a low-cost and reliable security solution.
- Demonstrate the integration of embedded systems with GSM communication.

---

## ✨ Features
- Real-time theft detection
- Instant SMS notification using GSM

## ⚙️ Working Principle
1. The sensor continuously monitors the protected area.
2. When motion or intrusion is detected, the microcontroller processes the signal.
3. The GSM module sends an SMS alert to the registered mobile number.
4. The buzzer sounds to indicate an intrusion.
5. The system returns to monitoring mode after the alert.
